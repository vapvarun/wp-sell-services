# Introduction to WP Sell Services

WP Sell Services transforms your WordPress site into a complete freelance marketplace like Fiverr or Upwork. Vendors can sell digital services, buyers can browse and purchase services, and you earn commission on every transaction.

## What is WP Sell Services?

WP Sell Services is a full-featured marketplace plugin that lets you create a service-based platform where freelancers and agencies can offer their expertise. The plugin handles everything from service listings to order management, messaging, payments, reviews, and disputes.

Perfect for building:
- Freelance marketplaces
- Gig platforms
- Service directories
- Agency networks
- Micro-job sites

## Key Features

### Service Management
- **Service Listings**: Vendors create detailed service pages with descriptions, pricing, and portfolios
- **Package Tiers**: Offer Basic, Standard, and Premium packages with different pricing and deliverables
- **Service Add-ons**: Extra services that buyers can add to their order (rush delivery, extra revisions, etc.)
- **Service Requirements**: Custom forms to collect buyer information before work starts

### Order System
- **Complete Order Workflow**: 10+ order statuses tracking every stage from payment to completion
- **Delivery Management**: Vendors submit deliverables, buyers request revisions
- **Deadline Extensions**: Request and approve delivery date changes
- **Order Messaging**: Built-in chat system for buyer-vendor communication

### Marketplace Features
- **Buyer Requests**: Buyers post project needs, vendors submit proposals
- **Reviews & Ratings**: 4-criteria rating system (Communication, Quality, Delivery, Overall)
- **Dispute Resolution**: Structured dispute system with mediation and refund options
- **Commission System**: Earn percentage or flat-rate commission on every sale

### Vendor Profiles
- **Vendor Levels**: Basic, Verified, and Pro vendor status
- **Portfolio Showcase**: Display past work and achievements
- **Vendor Dashboard**: Manage services, orders, earnings, and statistics

### Frontend Tools
- **6 Gutenberg Blocks**: Service Grid, Search, Featured Services, Categories, Seller Cards, Buyer Requests
- **12+ Shortcodes**: Display services, vendor profiles, dashboards, and more anywhere
- **Responsive Design**: Mobile-friendly templates for all pages
