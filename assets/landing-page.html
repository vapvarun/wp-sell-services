Built for Developers

Extensible architecture with REST API, hooks, and PSR-4 namespacing.

🔌

REST API First

21 full REST controllers with batch endpoint support. Build mobile apps or headless frontends on top of your marketplace.

⚙️

19+ Action Hooks

Filter and action hooks for every workflow step. Customize order flow, commissions, notifications, and vendor capabilities.

📦

PSR-4 Architecture

Clean namespace structure with dependency injection. Extend any service class without modifying core plugin files.

📄

Template Overrides

Copy any template to your theme's wp-sell-services/ folder and customize. Full control over every frontend view.

🔒

Secure by Default

Nonce verification, capability checks, input sanitization, and output escaping on every endpoint. WPCS compliant.

🏗️

CI/CD Ready

PHPUnit tests, PHPStan level 5, WPCS linting, and GitHub Actions CI pipeline. Production-grade from day one.
