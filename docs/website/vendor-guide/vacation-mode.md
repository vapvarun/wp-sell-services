# Vacation Mode

Vacation mode lets you temporarily pause new orders while you're away, without deactivating your account or losing your vendor status.

## What is Vacation Mode?

Vacation mode is a feature that:

- Prevents buyers from placing new orders
- Maintains your existing vendor profile
- Preserves your verification level
- Keeps your services visible (marked as unavailable)
- Allows you to fulfill existing orders

Use vacation mode when you need a break but plan to return.

## When to Use Vacation Mode

Enable vacation mode for:

- **Personal Vacation**: Holiday or travel
- **Illness**: Health recovery time
- **Family Emergency**: Unexpected personal situations
- **Workload Management**: Temporary overload
- **Business Break**: Planned hiatus
- **Seasonal Closure**: Off-season for your services

## How to Enable Vacation Mode

### Step-by-Step Activation

1. Log in to your vendor account
2. Navigate to **Dashboard → Settings → Vacation Mode**
3. Toggle **Enable Vacation Mode** to ON
4. Set vacation details:

![Vacation mode settings](../images/vendor-vacation-mode-settings.png)

#### Vacation Mode Settings

| Setting | Description | Required |
|---------|-------------|----------|
| **Start Date** | When vacation begins (can be today or future date) | Yes |
| **End Date** | When vacation ends (or leave blank for indefinite) | No |
| **Vacation Message** | Notice displayed to buyers on your profile | Yes |
| **Auto-Return** | Automatically disable vacation mode on end date | Optional |

### Sample Vacation Message

```
Thanks for visiting! I'm currently on vacation and will return
on March 15th. I'm not accepting new orders at this time, but
feel free to message me and I'll respond when I return.

For urgent inquiries, please contact: backup@example.com
```

Keep it professional, friendly, and informative.

## What Happens When Vacation Mode is Active

### For Buyers

When buyers visit your profile:

- **Vacation Banner**: Prominent notice at top of profile
- **Services Disabled**: "Order" buttons are disabled/hidden
- **Return Date**: Your expected return date is shown (if set)
- **Message Option**: Buyers can still send messages
- **Services Visible**: Your listings remain viewable

![Vacation mode buyer view](../images/vendor-vacation-mode-active.png)

### For Your Services

Your service listings:

- Remain published and visible in search
- Display "Vendor on Vacation" badge
- Cannot be purchased
- Maintain search ranking (frozen during vacation)
- Continue to accumulate favorites/saves

### For Existing Orders

Orders placed before vacation mode:

- **Continue Normally**: You can still fulfill active orders
- **Access Dashboard**: Full dashboard access maintained
- **Communicate**: Message buyers about ongoing orders
- **Deliver Work**: Submit deliverables as usual
- **Extensions**: May need to request delivery extensions

**Important**: Existing order deadlines don't automatically extend. Request extensions if needed.

## Setting an End Date

### Automatic Return

If you set an end date with auto-return enabled:

1. Vacation mode disables automatically on the end date
2. Services become available for purchase again
3. You receive email notification when reactivated
4. No action required from you

### Manual Return

If you don't set an end date (or disable auto-return):

- Vacation mode continues indefinitely
- You must manually disable it when ready
- No automatic reactivation

**Recommendation**: Always set an end date to avoid forgetting to reactivate.

## How to Disable Vacation Mode

### Early Return

If you return before your end date:

1. Navigate to **Dashboard → Settings → Vacation Mode**
2. Toggle **Enable Vacation Mode** to OFF
3. Confirm deactivation
4. Your services become available immediately

![Disable vacation mode](../images/vendor-vacation-mode-disable.png)

### What Happens on Return

When vacation mode ends:

- Services accept orders immediately
- Vacation banner removed from profile
- "Order" buttons re-enabled
- Email notification sent to you
- Buyers can purchase your services again

## Impact on Your Account

### Verification Level

Vacation mode does NOT affect:

- Your verification level (Basic/Verified/Pro)
- Account statistics
- Reviews and ratings
- Earnings history

Your level is protected during vacation.

### Search Ranking

During vacation:

- Services remain in search results
- Ranking position frozen (not improved or penalized)
- "On Vacation" filter may exclude you from some searches
- Ranking resumes normal calculation when you return

### Active Time Tracking

Account activity metrics:

- Activity timer pauses during vacation
- "Active seller" status maintained
- Verification criteria that include activity are paused
- Resumes tracking when vacation ends

## Vacation Mode Best Practices

### Before Activating

1. **Finish Active Orders**: Complete ongoing work when possible
2. **Communicate with Buyers**: Inform buyers of upcoming vacation
3. **Request Extensions**: Get approval for deadline changes
4. **Set Auto-Responders**: Prepare message templates
5. **Update Services**: Pause accepting rush orders beforehand

### During Vacation

1. **Check Messages**: Review messages periodically (optional)
2. **Monitor Active Orders**: Don't abandon existing commitments
3. **Maintain Deadlines**: Fulfill orders accepted before vacation
4. **Update End Date**: Adjust if plans change

### Upon Return

1. **Review Messages**: Respond to inquiries received during vacation
2. **Check Orders**: Address any pending order issues
3. **Update Profile**: Refresh profile if needed
4. **Announce Return**: Consider messaging recent buyers
5. **Manage Capacity**: Don't accept more orders than you can handle

## Alternatives to Vacation Mode

### Pause Individual Services

Instead of full vacation mode:

1. Navigate to **Dashboard → Services**
2. Click **Pause** on specific services
3. Other services remain available

Use this for partial availability.

### Extend Delivery Times

Temporarily increase delivery times:

1. Edit your services
2. Increase delivery days for each package
3. Save changes

Buyers can still order with longer wait times.

### Limit Order Queue

Some marketplaces let you:

- Set maximum active orders
- Pause orders when queue is full
- Automatically resume when queue clears

Check your dashboard for this feature.

## Vacation Mode Limitations

### Maximum Duration

Some marketplaces enforce limits:

- **Maximum vacation**: 30, 60, or 90 days
- **Frequency limits**: X vacations per year
- **Warning system**: Notifications before limits

Check your marketplace policies.

### Existing Order Obligations

Vacation mode does NOT:

- Cancel existing orders
- Extend order deadlines automatically
- Excuse late deliveries
- Prevent negative reviews for late work

You remain responsible for active orders.

### Repeated Use

Excessive vacation mode use may:

- Reduce buyer trust
- Lower search ranking over time
- Affect verification level eligibility
- Trigger admin review

Use vacation mode responsibly.

## Notifications

### Email Notifications

You receive emails when:

- Vacation mode activates
- Auto-return date approaches (3 days before)
- Vacation mode deactivates
- Vacation extends beyond expected return date

### Buyer Notifications

Buyers who favorited your services may receive:

- Notification when you return from vacation
- Alert that your services are available again

This drives return traffic to your profile.

## Mobile Management

Manage vacation mode on mobile:

- Enable/disable from mobile browser
- Responsive dashboard interface
- Push notifications (if app available)
- Quick toggle option

## Troubleshooting

### Vacation Mode Not Saving

- Check that end date is in the future
- Ensure vacation message is filled
- Clear browser cache
- Try different browser

### Services Still Show Order Button

- Wait 5 minutes for cache to clear
- Refresh the service page
- Verify vacation mode is enabled
- Contact support if persists

### Cannot Disable Vacation Mode

- Check if end date has passed
- Verify you're logged in as vendor
- Clear browser cache
- Contact support for assistance

### Existing Orders Affected

- Vacation mode shouldn't affect active orders
- Check order status in dashboard
- Contact support if orders are blocked
- Communicate with affected buyers

## Vacation Mode Analytics **[PRO]**

Pro marketplaces may track:

- Total vacation days used
- Impact on sales when returning
- Buyer retention after vacation
- Optimal vacation timing insights

## Tips for Successful Vacation

1. **Plan Ahead**: Announce vacation to buyers in advance
2. **Set Realistic End Dates**: Don't return early repeatedly
3. **Keep Commitments**: Finish active orders before leaving
4. **Brief Vacations**: Shorter breaks are better for business
5. **Strategic Timing**: Vacation during slow periods
6. **Professional Message**: Maintain professional tone
7. **Optional Contact**: Provide alternative contact if urgent
