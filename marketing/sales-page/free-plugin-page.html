How It Works

From installation to earning commission in four steps

Vendors Create Services

A 6-step wizard guides vendors through building detailed service pages with three pricing tiers, image galleries, videos, FAQs, and custom requirements.

Buyers Browse and Order

Buyers search by category, compare vendor ratings, select a pricing package with optional add-ons, and checkout. WooCommerce integration is optional for payment processing.

Order Workflow Runs Automatically

Requirements get collected, vendors deliver work with file attachments, buyers request revisions if needed, and orders auto-complete after approval.

You Earn Commission

Set a global commission rate (0-50%) or custom rates per vendor. When orders complete, earnings split automatically. Vendors request withdrawals through their dashboard.

Frequently Asked Questions

Common questions about WP Sell Services

Is WooCommerce required?
No. WP Sell Services works independently without WooCommerce. Your marketplace is fully functional for service listings, vendor management, order workflow, messaging, reviews, and dispute resolution. When WooCommerce is active, the plugin automatically enables checkout and payment processing with access to hundreds of payment gateways. The Pro version adds more options including Standalone mode with direct Stripe/PayPal/Razorpay integration.
Can I run a multi-vendor marketplace?
Yes. Any registered WordPress user can apply to become a vendor. You control vendor approval, set commission rates (global or per-vendor), moderate services, and oversee the entire marketplace.
How does the commission system work?
Set a global commission percentage (0-50%) in settings. When an order completes, commission is automatically calculated and split. Vendors see their available balance and request withdrawals. You can also set custom commission rates for individual vendors.
What payment gateways are supported?
When WooCommerce is active, all WooCommerce payment gateways work automatically, including Stripe, PayPal, Square, bank transfer, and hundreds more. The Pro version adds direct integrations for Stripe, PayPal, Razorpay, and offline payments that work independently without WooCommerce.
Can buyers post job requests?
Yes. Buyers can post project requests with budget range, description, and deadline. Vendors browse these requests and submit custom proposals. Buyers review proposals and accept the one they prefer.
How are disputes handled?
Buyers can open a dispute on any active order. Both parties submit evidence and messages. Administrators review details and can enforce resolutions including full refund, partial refund, additional revision, or note mutual agreement.
Does it work with my theme?
Yes. WP Sell Services is designed to work with any well-coded WordPress theme. All frontend templates can be overridden by copying them to your theme's wp-sell-services/ directory for customization.
Is it mobile-app ready?
Yes. The complete REST API with 20 controllers makes building iOS and Android apps straightforward. The batch endpoint allows mobile apps to execute multiple requests efficiently in a single HTTP call.
How do seller levels work?
Four levels: New Seller (default), Level 1 (5+ orders, 4.0+ rating), Level 2 (25+ orders, 4.5+ rating), and Top Rated (100+ orders, 4.8+ rating). Levels automatically update based on vendor performance metrics.
