## Pro Features

### 1. E-Commerce Platform Flexibility

**Free version integrates with WooCommerce when available.** Pro adds four alternatives so you choose what fits your needs — or use no e-commerce plugin at all.

#### Easy Digital Downloads (EDD)
Lightweight digital commerce platform with variable pricing and subscriptions. Lower overhead than WooCommerce.

**When to use:** Prefer a simpler e-commerce solution focused on digital products.

#### FluentCart
Modern, conversion-optimized checkout experience. Fast performance with streamlined interface.

**When to use:** Want higher conversion rates with cleaner checkout flow.

#### SureCart
Cloud-hosted checkout solution with built-in PCI compliance. Offload payment security to their servers.

**When to use:** Need PCI compliance without server-side payment processing.

#### Standalone Mode
No e-commerce plugin required. Built-in checkout system with direct payment gateway integration.

**When to use:** Want complete control without third-party plugin dependencies.

**How it works:**
1. Go to **Settings > General > E-Commerce Platform**
2. Select your preferred platform (or "Auto-detect" for automatic)
3. Configure platform-specific settings
4. Your marketplace switches checkout systems immediately

### 3. Wallet System Integrations

**Free version has built-in earnings tracking.** Pro adds four wallet provider integrations for automated vendor payouts.

#### Internal Wallet
Built-in wallet system with transaction history. Vendor earnings automatically credit to wallet on order completion. No external plugin required.

#### TeraWallet
WooCommerce wallet integration. Vendors use their WooCommerce wallet for marketplace earnings. Works with WooCommerce payment gateways.

#### WooWallet
WordPress wallet plugin support. Independent of WooCommerce. Vendors manage wallet balance through unified interface.

#### MyCred
Points-based reward system integration. Convert marketplace earnings to points. Vendors redeem points through MyCred system.

**How it works:**
1. Go to **Settings > Earnings > Wallet Provider**
2. Select your wallet system
3. Configure provider-specific settings
4. Earnings automatically credit to vendor wallets on completion
5. Vendors view balance and transaction history in dashboard

**Benefits:**
- Automated payouts (no manual withdrawal approvals)
- Transaction history and logs
- Multiple wallet types per marketplace
- Integration with existing wallet systems

### 7. Additional REST API Controllers

**Free version provides 20 REST API controllers.** Pro adds 4 more for Pro-specific features.

#### WalletController
**Base:** `/wp-json/wpss/v1/wallet`

**Endpoints:**
- `GET /balance` - Current wallet balance
- `GET /transactions` - Transaction history with pagination
- `POST /withdraw` - Request withdrawal
- `GET /providers` - List available wallet providers

#### PaymentController
**Base:** `/wp-json/wpss/v1/payments`

**Endpoints:**
- `POST /stripe/checkout` - Create Stripe checkout session
- `POST /paypal/checkout` - Create PayPal order
- `POST /razorpay/checkout` - Create Razorpay order
- `POST /offline/submit` - Submit offline payment proof
- `GET /status/:payment_id` - Check payment status

#### VendorAnalyticsController
**Base:** `/wp-json/wpss/v1/analytics`

**Endpoints:**
- `GET /overview` - Dashboard summary stats
- `GET /revenue` - Revenue data with date range filter
- `GET /orders` - Order analytics by status
- `GET /services` - Top services by revenue
- `POST /export` - Export analytics data (CSV/Excel)

#### StorageController
**Base:** `/wp-json/wpss/v1/storage`

**Endpoints:**
- `POST /upload` - Upload file to cloud storage
- `GET /download/:file_id` - Get signed download URL
- `DELETE /delete/:file_id` - Delete file from cloud storage
- `GET /providers` - List configured storage providers

**Benefits:**
- Build mobile apps with full Pro feature access
- Create custom integrations with external systems
- Automate analytics reporting
- Implement custom wallet workflows

## Frequently Asked Questions

### Is the free plugin required?
Yes. WP Sell Services Pro extends the free plugin. Both must be installed and active. Pro does not replace Free - it adds features via WordPress hooks.

### Do I still need WooCommerce with Pro?
No. Pro includes Standalone mode that runs without any e-commerce plugin. You can also use Easy Digital Downloads, FluentCart, or SureCart instead of WooCommerce.

### How does licensing work?
After purchase, you receive a license key. Enter it in **WP Sell Services > Settings > License** to activate Pro features and receive automatic updates. One license is valid for one site (or multiple sites with multi-site licenses).

### What happens if my license expires?
Your site continues working with all Pro features enabled. You will not receive plugin updates or support until you renew. No features are disabled.

### Can I upgrade from Free to Pro later?
Yes. Install Pro at any time alongside the free version. All existing services, orders, vendors, and settings are preserved. No migration or data transfer required.

### Which payment gateways work best?
**If using WooCommerce:** Any WooCommerce gateway (free or pro).
**If using Standalone mode:** Direct Stripe, PayPal, Razorpay, or Offline payments (Pro only).
**If using EDD/FluentCart/SureCart:** Their native payment gateways work automatically.

### How does the wallet system work?
When an order completes, vendor earnings (after commission) automatically credit to their chosen wallet. Vendors view balance, transaction history, and request withdrawals. You can use the built-in internal wallet or integrate with TeraWallet, WooWallet, or MyCred.

### Can I use multiple storage providers?
Yes. Configure multiple cloud storage providers and set a default. Different file types can route to different providers (e.g., images to S3, videos to Google Cloud).

### Does Pro include priority support?
Yes. All Pro licenses include priority email support with 24-hour response time during business days.

### What's your refund policy?
30-day money-back guarantee. If Pro doesn't meet your needs, contact us within 30 days for a full refund.
